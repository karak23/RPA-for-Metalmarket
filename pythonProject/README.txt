Ten program sciąga informacje o 1 uncjowych srebrnych monetach z strony https://www.metalmarket.eu/ i zapisuje dane o każdej monecie w pliku csv.
Aby odpalić ten projekt trzeba zrobić następujące kroki:

1.W terminale w Python envirounment jak naprzykład pycharm trzeba zainstalować moduły:
	pip install rpm
	pip install requests
	pip install beautifulsoup4
2. Następnie dodać main.py do projektu z python naprzykładk w pycharm i uruchomić
